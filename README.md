##   About me
<img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="35"> I'm a Software Engineer and a self-taught developer.

I’m currently focusing on C# and .NET to build desktop applications, but I also have experience with Laravel, JavaScript, and MySQL, both in professional projects and personal experiments.

I enjoy the satisfaction of solving problems on my own and seeing how everything fits together like a puzzle within a real project. For me, technology should be more than just functional—it should deliver tangible value to its users.
